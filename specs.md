To build Calcy, we will synthesize the functional depth of calculator.net and WolframAlpha with the high-speed, modern "SaaS" aesthetics of 2026. By using a Vanilla HTML/JS stack, you ensure the site is incredibly fast, easy to crawl for AdSense, and requires zero maintenance.1. The UX Strategy (The "Calcy" Philosophy)From calculator.net: We take the Categorization. It’s the "everything" store.From wolframalpha.com: We take the Command-Line Speed. Users should be able to type "mort" and hit Enter to get to the Mortgage calculator immediately.From online-calculator.com: We take the Simplicity. Large, clear inputs that are mobile-friendly.2. The Final Tech SpecsComponentspecificationFoundationSingle-file index.html structure (for maximum speed).LogicVanilla ES6 JavaScript. Each calculator is a JS object containing its own formula, template, and meta (SEO text).StylingTailwind CSS (CDN-free build). Use a "Glassmorphism" palette: semi-transparent cards, soft shadows, and high-contrast typography.RoutingHash-based Navigation (calcy.tools/#mortgage). This allows users to bookmark specific calculators without needing a backend.3. The 4-Zone AdSense ArchitectureThis layout maximizes revenue without annoying the user (keeping you "AdSense Safe").Zone A (Header): Adaptive banner below the search bar but above the tool.Zone B (Side-Rail): A fixed "Sticky" sidebar on the right for 300x600 skyscraper ads. (Desktop only).Zone C (Native Contextual): An ad placed exactly between the calculation inputs and the results. This has the highest click-through rate.Zone D (Footer): A standard horizontal anchor ad.4. The Core Features (MVP)The AI should build these four "High-Revenue" modules first:Financial Suite: Mortgage with Amortization, Compound Interest, and a "Salary to Hourly" converter.Health Suite: BMI, TDEE (Total Daily Energy Expenditure), and Water Intake.Math Suite: Percentage Change (the most searched math term), Scientific Calc, and Fraction-to-Decimal.Unit Suite: "Smart" conversion (one box that handles Length, Weight, and Temp based on what you type).5.
